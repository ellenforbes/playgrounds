# Playgrounds
A web application for discovering and exploring playgrounds and finding childrens events and playgroups (Hunter Region and Brisbane).

🌐 **Live Demo:** [playgrounds-iota.vercel.app](https://playgrounds-iota.vercel.app)

## Features
- 🗺️ Interactive map interface for exploring playgrounds
- 🔍 Search and filter functionality
- 📊 Comprehensive playground information and details
- 🌐 Event scraping from multiple sources
- 📱 Responsive design for mobile and desktop
- ⚡ Still working on optimised performance :)

## Tech Stack
### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Modern styling with custom styles
- **JavaScript** - Interactive functionality

### Backend
- **Python** - Server-side logic and web scraping and API connections
- **Supabase** - SQL Database (PostGIS enabled) for holding playground and events data

### Infrastructure
- **Vercel** - Deployment and hosting
- **GitHub Actions** - CI/CD pipeline
- **Resend** - Email Service
